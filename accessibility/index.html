The fine print

Accessibility

Effective date: 2026-07-15

Everlane Realty wants everyone to be able to use this website, including people with disabilities. We build and maintain the site with accessibility in mind and treat it as ongoing work, not a one-time checkbox.

Our standard

We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 at Level AA — the standard widely used as the benchmark for accessible websites.

What we’ve built in

  • Readable color contrast. Every text-and-background color pair used on the site is computed against the WCAG AA contrast thresholds.
  • Clean, semantic HTML. Pages are static, server-rendered HTML with proper headings and landmarks, so they work well with screen readers and other assistive technology.
  • Keyboard navigation. The site can be operated with a keyboard, and a “skip to content” link lets keyboard and screen-reader users jump past the navigation.
  • Labelled forms. Form fields have associated, visible labels, and grouped choices use proper field grouping.
  • Descriptive images. Meaningful images include alternative text.
  • Reduced motion. The site respects your browser or device “reduce motion” setting.

Known limitations

The property-search and map tools embedded on this site are provided by a third-party MLS/IDX vendor and are rendered as vendor-controlled components that we cannot fully modify. As a result, those embedded widgets may not fully meet WCAG 2.1 AA. We’re aware of this, we monitor it, and we’ll work with the vendor as their tools improve. In the meantime, if a listing or search tool is hard to use, contact us and we’ll get you the same information directly.

Ongoing work

Accessibility is never finished. We review the site as it changes and fix issues we find or that are reported to us.

Need help, or found a problem?

If you have trouble using any part of this site, or you need information from it in a different format, please tell us — we’ll help, and we’ll get you what you need directly.
